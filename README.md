##起動方法
Java実行環境をインストールし、  
ClientManager.batをクリックして起動してください。  

##操作方法
宿泊者名と到着日時、出発日時、人数を入力し、  
利用明細も入力してください。  
保存ディレクトリを確認し、入力反映ボタンを押すと、  
指定されたディレクトリにExcelファイルが出力されます。  

##右下の顧客表示ボタンを押すと、今まで登録された宿泊者の利用履歴が表示されます。  
宿泊者名をクリックすると、その人が泊まった日付が表示されます。  
そのまま日付をクリックすると、利用明細の一覧が表示されます。  
削除ボタンを押すと選択されている宿泊日時の利用明細を削除します。  
編集ボタンを押すと選択されている宿泊日時の利用明細編集画面が開きます。  
操作は明細作成画面と同じです。
